# Pretentious meal description

## Usage
### 1. Navigate to ``pretentious-menu`` directory
  ```bash
  cd pretentious-menu
  ```
### 2. Install dependencies
  ```bash
  npm install
  ```
### 3. Run app
#### Node version
  ```bash
  npm -s start
  ```
  or
  ```bash
  node index.js
  ```
#### Client version
##### 1. Navigate to ``client`` directory
  ```bash
  cd client
  ```
##### 2. Run script
  ```bash
  npm run ui
  ```
*This will open the visual app in a new browser tab.*
